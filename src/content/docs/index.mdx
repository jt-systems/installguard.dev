        distance-1 names.
      </p>
    </article>
    <article class="ig-card">
      <div class="ig-card-eyebrow">Provenance</div>
      <h3 class="ig-card-title">Publisher &amp; account changes</h3>
      <p class="ig-card-body">
        Maintainer churn, brand-new npm accounts, and missing or mismatched
        provenance attestations &mdash; npm <span class="ig-chip">--provenance</span>
        bundles and PyPI <span class="ig-chip">PEP 740</span> Trusted Publisher attestations.
      </p>
    </article>
    <article class="ig-card">
      <div class="ig-card-eyebrow">Execution</div>
      <h3 class="ig-card-title">Install-time lifecycle scripts</h3>
      <p class="ig-card-body">
        Deny-by-default policy on <span class="ig-chip">preinstall</span>,
        <span class="ig-chip">install</span>, <span class="ig-chip">postinstall</span>
        and PyPI <span class="ig-chip">setup.py</span>, with a curated allowlist for
        native-binary packages, plus pattern-based detection of obfuscated payloads
        across both shell and Python.
      </p>
    </article>
    <article class="ig-card">
      <div class="ig-card-eyebrow">Velocity</div>

        <span class="ig-chip">bin</span> entries or scripts vs. the previous

<section class="ig-section not-content" style="margin-top: clamp(4rem, 8vw, 6.5rem);">
  <span class="ig-eyebrow">Engineered for the install path</span>
  <h2 class="ig-section-title">A gate, not a scanner.</h2>
  <p class="ig-section-lede">
    Static Rust binary. Reads lockfiles directly. Never resolves, never installs,
    never executes. Designed to slot in front of <code class="ig-chip">npm install</code>
    without slowing it down.
  </p>
  <div class="ig-grid ig-grid--3">
    <article class="ig-card">
      <div class="ig-stat-num">~2s</div>
      <h3 class="ig-card-title">Sub-CI latency</h3>
      <p class="ig-card-body">
        Warm-cache scan of a 1,000-package lockfile in under two seconds.
        No daemon, no JVM, no Python runtime.
      </p>
    </article>
    <article class="ig-card">
      <div class="ig-stat-num">0</div>
      <h3 class="ig-card-title">Side effects</h3>
      <p class="ig-card-body">
        Reads <span class="ig-chip">package-lock.json</span>,
        <span class="ig-chip">pnpm-lock.yaml</span>,
        <span class="ig-chip">yarn.lock</span>,
        <span class="ig-chip">uv.lock</span>,
        <span class="ig-chip">poetry.lock</span>, and pinned
        <span class="ig-chip">requirements.txt</span> directly.
        Never installs, never executes lifecycle scripts. Network
        calls (registry metadata, OSV, deps.dev, Scorecard, PyPI
        Integrity API) are opt-out per provider, and
        <span class="ig-chip">--frozen</span> runs entirely from
        <span class="ig-chip">installguard.lock</span> — zero sockets.
      </p>
    </article>
    <article class="ig-card">
      <div class="ig-stat-num">100<span style="color:#6ee7b7;font-size:0.7em">%</span></div>
      <h3 class="ig-card-title">Auditable verdicts</h3>
      <p class="ig-card-body">
        Every block is recorded in <span class="ig-chip">installguard.lock</span>
        with the exact signals that produced it. Deterministic re-runs.
        JUnit XML, JSON, GitHub PR sticky-comment.
      </p>
    </article>
  </div>
</section>
